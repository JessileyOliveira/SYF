# Search Your Movie

> Aplicativo para busca de filmes

## Resumo

Aplicativo desenvolvido em react-native que faz a busca de filmes e mostra o nome e o "poster".

> OBS:. Crie sua conta no [themoviedb](https://www.themoviedb.org/) e receba seu token. Para incluir no projeto vรก em `./src/services/API.ts`.
